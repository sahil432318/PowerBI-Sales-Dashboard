# 📊 Sales Dashboard using Power BI

## 📌 Project Overview
This project focuses on analyzing sales performance using Power BI to generate meaningful business insights and interactive visualizations. The dashboard helps track sales trends, profit distribution, customer behavior, and regional performance for better decision-making.

## 🛠️ Tools Used
- Power BI
- SQL
- Excel

## 📈 Key KPIs
- 💰 Total Sales
- 📊 Total Profit
- 👥 Customer Count

## 🚀 Dashboard Features
- Region-wise Sales Analysis
- Product Performance Analysis
- Interactive Filters & Slicers
- Profit Trend Visualization
- Category-wise Revenue Insights
- Dynamic Charts and Cards

## 🔍 Key Insights
- The **West Region** generated the highest sales.
- The **Technology Category** contributed the maximum revenue.
- Interactive dashboards help identify top-performing products and regions quickly.
